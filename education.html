Course Descriptions

CSE 4311 - Neural Networks and Deep Learning

Introduced neural networks and deep learning concepts, covering perceptrons, single-layer and multi-layer neural networks, TensorFlow, Keras, convolutional neural networks (CNNs), transfer learning, and deep learning methods for object recognition and detection in images. Explored sequential learning models for text analysis, auto-encoders, and generative adversarial networks (GANs). Required a strong programming and algorithmic background, along with knowledge of linear algebra.

CSE 4309 - Fundamentals of Machine Learning

Introduced core machine learning concepts, including naive Bayes classifiers, linear regression, linear classifiers, neural networks, backpropagation, kernel methods, decision trees, feature selection, clustering, and reinforcement learning. Emphasized a strong programming background, alongside knowledge of linear algebra, probability theory, and statistics.

CSE 4308 - Artificial Intelligence

Introduced foundational artificial intelligence techniques, including heuristic search, deduction, learning, problem-solving, knowledge representation, uncertainty reasoning, and symbolic programming with LISP. Explored applications such as intelligent agents, data mining, natural language processing, machine vision, planning, and expert systems.

CSE 4305 - Compilers for Algorithmic Languages

Reviewed programming language structures, translation, and storage allocation. Studied the theory and practice of compiler construction, focusing on parsing, intermediate code generation, local optimization issues (e.g., register allocation), data-flow analysis, and global optimization.

CSE 3330 - Database Systems and File Structures

Covered database system architecture and file structures, including indexing, hashing, and B+-trees. Studied the relational model and algebra, SQL database language, Entity-Relationship data modeling, functional dependencies, and basic normalization.

CSE 3320 - Operating Systems

Explored the functions and components of operating systems, including process synchronization, job scheduling, memory management, file system protection, and deadlocks. Examined related system software, such as loaders, linkers, assemblers, and windowing systems.

CSE 3318 - Algorithms & Data Structures

Focused on the design and analysis of algorithms, emphasizing data structures. Covered classical algorithm design techniques, including sorting, searching, and operations on data structures such as hash tables, trees, graphs, and strings. Explored dynamic programming, greedy approaches, and techniques for analyzing lower and upper bounds on problems and algorithms.

CSE 3310 - Fundamentals of Software Engineering

Covered software engineering principles, processes, and techniques, with a focus on functional analysis and design methods. Included configuration management, implementation strategies, and testing. Emphasized practical application through a team project.

CSE 3302 - Programming Languages

Introduced and analyzed important concepts across various programming languages. Examined formalisms for specifying language syntax and semantics, and explored programming paradigms including algorithmic, functional, logic, and object-oriented approaches.

CSE 2312 - Computer Organization & Assembly Language Programming

Explored computer organization from a software perspective, including instruction set architectures, memory addressing, integer and floating-point representation and arithmetic, instruction pipelining, cache, memory virtualization, and I/O. Examined the relationship between higher-level programming languages and assembly language, as well as instruction set architecture.

CSE 1325 - Object-Oriented Programming

Covered object-oriented programming concepts including class diagrams, collection classes, generics, polymorphism, and reusability. Involved extensive programming projects, including graphical user interfaces and multithreading.
