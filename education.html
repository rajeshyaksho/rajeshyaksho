Course Descriptions

CSE 4311 - Neural Networks and Deep Learning

  • Introduced neural networks and deep learning concepts, including perceptrons, single-layer and multi-layer networks, TensorFlow, and Keras.
  • Covered CNNs, transfer learning, and deep learning methods for object recognition and detection, as well as sequential learning models for text analysis and generative adversarial networks (GANs). Required strong programming and algorithmic skills.

CSE 4309 - Fundamentals of Machine Learning

  • Covered core machine learning concepts such as naive Bayes classifiers, linear regression, and neural networks, with an emphasis on backpropagation and kernel methods.
  • Included decision trees, clustering, and reinforcement learning, requiring a solid programming background and knowledge in linear algebra, probability theory, and statistics.

CSE 4308 - Artificial Intelligence

  • Introduced foundational AI techniques including heuristic search, deduction, learning, and problem-solving. Covered knowledge representation, uncertainty reasoning, and symbolic programming with LISP.
  • Explored practical applications such as intelligent agents, data mining, natural language processing, machine vision, planning, and expert systems.

CSE 4305 - Compilers for Algorithmic Languages

  • Reviewed programming language structures and compiler construction theory, focusing on parsing, intermediate code generation, and local optimization issues like register allocation.
  • Studied data-flow analysis and global optimization techniques, essential for effective compiler design.

CSE 3330 - Database Systems and File Structures

  • Covered database system architecture and file structures, including indexing, hashing, and B+-trees.
  • Studied the relational model, SQL database language, Entity-Relationship modeling, functional dependencies, and basic normalization techniques.

CSE 3320 - Operating Systems

  • Explored operating system functions and components, including process synchronization, job scheduling, memory management, and file system protection.
  • Reviewed system software like loaders, linkers, and assemblers, and examined concepts related to deadlocks and system software.

CSE 3318 - Algorithms & Data Structures

  • Focused on the design and analysis of algorithms with an emphasis on data structures. Covered classical techniques such as sorting, searching, and operations on hash tables, trees, and graphs.
  • Explored dynamic programming, greedy algorithms, and methods for analyzing algorithmic complexity and problem-solving strategies.

CSE 3310 - Fundamentals of Software Engineering

  • Covered software engineering principles and processes, including functional analysis, design methods, configuration management, and implementation strategies.
  • Emphasized practical application through team projects, focusing on real-world software development and testing methodologies.

CSE 3302 - Programming Languages

  • Introduced and analyzed key concepts across various programming languages, including syntax, semantics, and paradigms.
  • Explored different programming paradigms such as algorithmic, functional, logic, and object-oriented approaches.

CSE 2312 - Computer Organization & Assembly Language Programming

  • Explored computer organization, including instruction set architectures, memory addressing, and arithmetic operations. Covered topics like instruction pipelining, cache, memory virtualization, and I/O.
  • Examined the relationship between high-level programming languages and assembly language, focusing on instruction set architecture.

CSE 1325 - Object-Oriented Programming

  • Covered object-oriented programming concepts such as class diagrams, collection classes, generics, and polymorphism.
  • Involved extensive programming projects, including graphical user interfaces and multithreading, to apply and solidify concepts.
